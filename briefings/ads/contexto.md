# Módulo ADS — Piuka

## O que cobre

Meta Ads e Google Ads do mês: verba, objetivo de campanha, criativos, copy de anúncio. Output vai para Ana (gestora das campanhas pagas).

Seções de origem no instalador: Bloco 4 (canais/verba — parte ADS), seção 2.2 (copy Meta Ads).

---

## Perguntas de extração

### Bloco 1 — Verba e objetivo

```
→ Qual é a verba total para Meta Ads este mês?
→ Google Ads está ativo este mês? Se sim, qual a verba?
→ Qual é o objetivo principal das campanhas pagas: conversão, awareness, tráfego, remarketing?
→ Tem campanha de remarketing ativa ou vai criar uma nova?
```

### Bloco 2 — Criativos

```
→ Quais criativos vão para os anúncios? (produtos, fotos disponíveis, formato — feed, story, reels)
→ Tem foto e vídeo disponíveis ou o time precisa produzir?
→ Qual produto ou coleção vai ser o foco dos anúncios?
→ Tem restrição de criativo? (ex: "não usar foto X", "não mostrar preço no criativo")
```

### Bloco 3 — Copy dos anúncios

```
→ Tom do anúncio: urgência, benefício, emocional ou misto?
→ Tem desconto ou oferta específica para os anúncios?
→ Qual o CTA principal? (ex: "Comprar agora", "Ver coleção", "Aproveitar")
→ Tem algum copy ou headline de campanha anterior que funcionou bem e quer replicar o tom?
```

## Situações comuns

**Ana não sabe a verba exata:**
"⚠️ Verba Meta Ads: a confirmar. Usando [valor do mês anterior] como referência provisória."

**Sem criativo disponível:**
"Entendido. Vou sinalizar no output que o time precisa produzir criativo antes de subir a campanha."

**Tom indefinido:**
"Para este mês, qual resultado você quer priorizar: mais cliques ou mais conversões? Isso define o tom."
