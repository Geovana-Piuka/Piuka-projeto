## COMO USAR ESTE AGENTE

Este arquivo é o "cérebro da marca" — carregado como contexto base antes de qualquer outro agente da Piuka executar uma tarefa.

**Todo agente da Piuka começa aqui.** O agente de briefing usa este contexto. O agente de copy usa este contexto. O agente de e-mail usa este contexto. Sem ele, qualquer output sai genérico e a Ana reprova.

**Como alimentar:**
- Sessão de extração com Ana (2-3h) — preencher os campos `[CAMPO_ANA]`
- A cada campanha aprovada: adicionar entrada na seção HISTÓRICO DE CAMPANHAS
- A cada reprova com comentário da Ana: adicionar regra na seção CRITÉRIOS DE APROVAÇÃO
- Geovana é responsável por manter esse arquivo atualizado com cada ciclo

`[CAMPO_ANA — erros específicos que Ana não tolera — lista dos "jamais faça isso"]`

### Filtro de revisão (usar antes de qualquer entrega para Ana)

Antes de qualquer copy, legenda, e-mail ou texto chegar para aprovação da Ana, verificar:

- [ ] O texto soa como a Piuka ou como qualquer outra marca?
- [ ] Tem erro de português ou digitação?
- [ ] O produto está nomeado com exatidão (não "nossa coleção" mas "Coleção Letras")?
- [ ] O CTA é direto e único?
- [ ] A urgência, se houver, tem fundamento real (data, estoque, cupom real)?
- [ ] O tom está correto para o canal específico?
- [ ] Alguma palavra ou expressão proibida está presente?
