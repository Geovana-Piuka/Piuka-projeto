# SOP — Isabelle | Designer
**Versão:** 1.0 | **Piuka × Styem**

---

## Início do mês

1. Receber o sub-doc da Isabelle — está no briefing do mês aprovado pela Ana
2. Abrir a tabela de prioridade de entrega — está ordenada por data limite
3. Para cada arte: ler o briefing visual correspondente na seção 2.5 do briefing
4. Criar os cards no Trello com data limite e marcar como "Em andamento"
5. Dúvida antes de começar: consultar o briefing visual. Se não estiver coberta, consultar Marcela — não Ana diretamente

---

## Durante o mês

**Para cada arte:**

1. Ler o briefing visual completo antes de abrir o arquivo
   - Formato e dimensões
   - Headline obrigatória
   - Hierarquia visual (1º, 2º, 3º elemento)
   - O que é mandatório vs. opcional
   - Referência visual descrita
   - O que não pode

2. Criar a arte com base no briefing — não inventar fora do que está descrito

3. Antes de entregar para Ana: usar o Gemini para verificar se o briefing visual foi respeitado
   - Colar o briefing + descrever o que foi criado → pedir checagem
   - Se houver discrepância: corrigir antes de entregar

4. Entregar no Trello com prazo respeitado
5. Mover o card para "Aprovar | Arte" com foto/arquivo da peça anexado

**Para fotos de sessão:**
- As sessões de mídia e humanizadas são fixas mensais — ver datas no briefing fotográfico (seção 2.5B)
- Sessão extra de campanha: ver moodboard e direcional no briefing fotográfico
- Prazo crítico de aprovação de fotos: consta no briefing — respeitar para não travar o calendário de artes

---

## Uso do Freepic

- Para variações de fundo, texturas, elementos decorativos: usar Freepic antes de pedir nova sessão fotográfica
- Briefing da campanha indica se o produto precisa de foto nova ou se variação digital resolve
- Dúvida: consultar Marcela. Decisão de sessão extra: escalar para Ana com a pergunta já estruturada ("preciso de foto nova ou dá para resolver com Freepic? O produto X ainda não tem foto nesse estilo")

---

## Quando escalar para Ana

- Quando o briefing visual estiver incompleto para uma arte específica e o prazo não permitir esperar
- Quando houver conflito de identidade visual não coberto pelo briefing

**Não escalar para Ana:**
- Escolha entre fontes dentro da paleta da marca
- Dúvida sobre dimensão — está no briefing ou consultar Marcela
- Decisão de Freepic vs. foto — consultar Marcela primeiro

---

## Regras de ouro

- Briefing visual não é sugestão — é o que a Ana aprovou. Desvio do briefing = reprova garantida
- Prazo estourado: avisar Marcela com 2 dias de antecedência, não no dia
- Uma arte entregue no prazo e dentro do briefing passa de primeira. Duas reprovações seguidas = conversar com Ana sobre o que está faltando no briefing
