# SOP — Amanda | Social Media
**Versão:** 1.0 | **Piuka × Styem**

---

## Início do mês

1. Receber o sub-doc do Amanda — está no briefing do mês aprovado pela Ana
2. Abrir o calendário editorial — está em formato de tabela com data, tipo, canal, formato, copy e orientação visual
3. Criar os cards no Trello com base no calendário — um card por data de publicação
4. Qualquer dúvida sobre o briefing: consultar o sub-doc antes de perguntar para Ana ou Marcela

---

## Durante o mês

**Para cada post:**

1. Abrir o card do Trello correspondente
2. Usar a copy que está no sub-doc como base
3. Se quiser ajustar o texto: usar o Agente de Copy no Gemini antes de postar
   - Colar o texto → pedir revisão para a voz da Piuka → usar a versão revisada
   - Nunca postar texto direto do ChatGPT genérico — sempre passar pelo agente
4. Aplicar a orientação visual que está no card
5. Postar no horário previsto
6. Mover o card para "Enviados/Concluídas" após publicar

**Para posts de campanha principal:**
- Stories do dia de lançamento: usar a sequência de 3 frames gerada no briefing
- Feed: usar uma das 3 opções de copy geradas — escolha livre dentro das opções

**Para dias sazonais:**
- O card está marcado com TIPO: SAZONAL
- Usar o conteúdo evergreen ou direcional indicado no card
- Não deixar vazio — sazonal é conteúdo, não dia de folga

---

## Quando escalar para Ana

- Quando o briefing não tiver orientação para uma data específica
- Quando uma campanha for alterada depois de publicada e o calendário precisar ser refeito
- Quando houver conflito entre duas datas no mesmo dia e o sub-doc não definir prioridade

**Não escalar para Ana:**
- Dúvidas sobre copy que o agente de revisão consegue resolver
- Dúvidas sobre hashtag — escolha livre dentro do que faz sentido para o canal
- Escolha entre as opções de copy geradas — a decisão é sua

---

## Regras de ouro

- Nunca postar texto sem passar pelo agente de revisão — economiza uma reprova da Ana
- O calendário é estrutura, não limite — se tiver uma ideia boa de sazonal, registra no card e avisa Marcela
- Stories de lançamento: sempre usar a sequência de 3 frames — não improvisar no dia de campanha
