# Módulo Fotos — Piuka

## O que cobre

Sessões fotográficas do mês: as duas fixas mensais (mídia e humanizada) e sessões extras de campanha. Output vai para o fotógrafo terceirizado via Marcela.

Seções de origem no instalador: Sub-bloco 4C, seção 2.5B.

**Sessões fixas mensais (sempre existem, não perguntar se vão acontecer — só confirmar datas):**
- Fotos de mídia: produto puro, sem corpo, fundo neutro
- Fotos humanizadas: produto no corpo, sem rosto, fundo neutro ou ambiente

---

## Perguntas de extração

### Bloco 1 — Sessões fixas

```
→ Qual é a data da sessão de fotos de mídia este mês?
→ Qual é a data da sessão de fotos humanizadas este mês?
→ Quais produtos vão ser fotografados nas sessões fixas?
→ Tem prazo crítico de aprovação das fotos (alguma arte que trava sem a foto)? Qual data?
```

### Bloco 2 — Sessão extra de campanha (se houver)

```
→ Tem campanha específica que precisa de sessão extra este mês?
→ Se sim:
   • Qual campanha / motivo?
   • Tem moodboard? (S/N — se sim, pedir link ou descrição)
   • Qual é o direcional de estilo? (paleta, ambiente, styling)
   • Vai ter influencer ou modelo na sessão?
   • Quais produtos entram?
   • Quantas fotos e em qual formato?
   • Qual o prazo crítico de aprovação?
```

## Situações comuns

**Data das sessões fixas não confirmada:**
"⚠️ Data sessão de mídia: a confirmar. Quando souber, atualizar o briefing fotográfico."

**Sessão extra sem moodboard:**
Sinalizar ⚠️ e não avançar sem direcional mínimo: "Preciso de pelo menos 1 referência visual ou descrição de paleta para o fotógrafo."

**Prazo apertado:**
"Com o prazo, qual sessão é prioritária? Vou colocar no output que a aprovação de fotos precisa ser antes de [data] para não travar a arte."
