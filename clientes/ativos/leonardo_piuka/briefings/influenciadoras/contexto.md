# Módulo Influenciadoras — Piuka

## O que cobre

Influencers confirmadas do mês e afiliadas: produto enviado, prazo de kit, briefing de conteúdo, missão mensal. Output vai para Pietra (gestão de influencers e afiliadas).

Seções de origem no instalador: Sub-bloco 4B, seção 2.4B.

---

## Perguntas de extração

### Bloco 1 — Influencers do mês

```
→ Tem influencers confirmadas para este mês? (padrão: 3-4 recorrentes)
→ Tem influencer nova este mês ou são as recorrentes?
→ Vai ter collab de coleção (produto exclusivo com o nome dela)?

Para cada influencer confirmada, coletar:
   • Nome real + nome artístico
   • @handle Instagram (OBRIGATÓRIO — usado em /imagens-piuka e /emails-piuka)
   • @handle TikTok (se tiver — opcional)
   • Nicho / público
   • Estilo visual em 1 linha (ex: "editorial minimalista neutro", "maximalista colorida")
   • O que ela vai receber neste mês (produto, kit, coleção)
   • Prazo de entrega do kit
   • O que ela vai publicar (formato — stories / reels / feed)
   • Prazo de publicação esperado
   • Para ela a Piuka vai montar assessoria de briefing completo? (S/N)
```

### Bloco 2 — Afiliadas

```
→ Qual é a missão de posts diários deste mês?
→ Tem competição de venda prevista? Qual é o prêmio?
→ Tem conteúdo especial para o grupo do WhatsApp das afiliadas?
→ Alguma afiliada merece destaque este mês?
```

## Situações comuns

**Ana não tem todas as influencers confirmadas:**
"Quais já estão certas? As demais posso deixar como ⚠️ pendente para completar depois."

**@handle não informado:**
Não avançar sem o @handle Instagram. "O @handle do Instagram é obrigatório — a skill /imagens-piuka vai precisar para gerar as peças da collab."

**Afiliadas sem missão definida:**
"⚠️ Missão das afiliadas: a definir. Quer que eu sugira algo com base na campanha principal do mês?"
